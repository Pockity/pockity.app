# Pockity Website

This website is built using [Eleventy](https://www.11ty.dev/).

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm start
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Project Structure

- `pages/`: Static pages.
- `_posts/`: Blog posts.
- `_includes/`: Template partials.
- `_layouts/`: Page layouts.
- `_sass/`: SCSS styles (includes LaunchKit theme).
- `assets/`: Static assets (images, fonts, js, videos).
